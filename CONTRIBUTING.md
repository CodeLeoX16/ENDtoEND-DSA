# Contributing Guide

Thank you for your interest in contributing to this DSA repository!

## How to Add a New Problem

1. **Choose the appropriate category folder** based on the problem's primary concept
2. **Create a new folder** for your problem inside the category folder
3. **Use the template** from `TEMPLATE.md` to document your solution
4. **Include your solution code** in the appropriate language
5. **Update the category README** to list your new problem

## Naming Conventions

- **Folder names**: Use PascalCase or hyphen-separated (e.g., `TwoSum` or `two-sum`)
- **File names**: Use lowercase with appropriate extensions (e.g., `solution.py`, `solution.java`)

## Code Quality Guidelines

- Write clean, readable code with comments
- Include time and space complexity analysis
- Add test cases in your README
- Follow the language-specific best practices

## Updating Progress

Don't forget to update `PROGRESS.md` with your daily achievements!
